These data were derived from Open Street Map land polygons downloaded on 2016-09-14 from:
http://openstreetmapdata.com/data/land-polygons

Processing steps
1) Select by hand British Isles (Great Britain, Ireland and small islands)
2) Save as new layer using Brish Nationla Grid coordinate system (EPSG:27700)
3) Add Area column using QGIS field calculator $area variable
